# Plan: Test app op telefoon

## Doel
De SA Chat app testen op je telefoon terwijl de server op je computer draait.

## Opties

### Optie 1: Zelfde WiFi (Makkelijkste)
1. Zorg dat telefoon en computer op hetzelfde WiFi-netwerk zitten
2. Vind je computer's IP-adres:
   - **Windows**: Open cmd, type `ipconfig` → zoek "IPv4-adres" (meestal `192.168.x.x`)
3. Start de server: `node server.js`
4. Open op telefoon: `http://[jouw-ip-adres]:3000`

### Optie 2: ngrok (Beste voor extern testen)
1. Download ngrok van https://ngrok.com
2. Start server: `node server.js`
3. Start ngrok: `ngrok http 3000`
4. Open de URL die ngrok geeft op je telefoon

### Optie 3: USB debugging (Android)
1. Zet USB debugging aan in Android ontwikkelaarsopties
2. Verbind telefoon met USB
3. Gebruik Chrome DevTools → Port forwarding

## Aanbevolen
**Optie 1** voor snel testen op hetzelfde netwerk.
